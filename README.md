# CodeAlpha_Build_a_Calculator
Built an calculator using HTML, CSS, and JavaScript. Supports all basic arithmetic operations with real-time results, dark/light theme toggle, keyboard input, mobile responsiveness, and persistent calculation history using localStorage.

## Features
- Basic arithmetic operations (+, −, ×, ÷)
- Dark and Light theme toggle
- Keyboard support
- Mobile responsive design
- Calculation history saved using localStorage

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone the project
2. Open the folder in VS Code
3. Open `index.html` using Live Server or in a browser
